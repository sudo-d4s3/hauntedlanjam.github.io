ABOUT

What is the Haunted LAN Jam?

The Haunted LAN Jam is an independent LAN Party put on by local gamers, hackers, and game jammers. The goal is not only to connect the community but also a way to show off demos and homebrew games on top of playing some video games.

What is a LAN Party?

A LAN party is a party where a bunch of gamers play games in the same room on the same network.

What Games Will Be Played?

Thats up to you and the other players!

To help facilitate we will be running:

  • A cache server with a handful of games pre-cached
  • A file server with freeware games and other goodies
  • Dedicated Game Servers

    • You can learn more below.

What Is A Demo?

A Demo is a very small audio/visual computer art project.

There are different catagories for demos such as (but not limited to) 4k intros, where the program is only 4 kilobytes, and 64k intros, where the program is only 64 kilobytes.

What Is A GameJam?

A Game Jam is an event where a bunch of game developers get together and make a game.


This can be either a competition or collaboration. We tend to like a combination of the two.

Servers We'll Be Running

Cache

Server

When you try and download a game through steam this server will download it on your behalf and then send it to you and anyone else who tries to download the game at lan speeds.

This should help prevent us from thrashing our host's network.

File

Server

We figure this isn't the year of the Linux desktop (sadly) so we are running an SMB file server for maximum compatibility with Windows machines.

This will have a bunch of freeware games that we won't get in trouble for distributing.

Game

Servers

We have chosen Pelican to be our game panel of choice this year.

This will let us monitor server health and kill/rebuild servers quickly if needed.
